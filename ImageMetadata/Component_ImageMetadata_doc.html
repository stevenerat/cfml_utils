com.stevenerat.util.ImageMetadata
Component ImageMetadata (Image Metadata Utility)


Custom enhancement for ColdFusion's built-in EXIF and IPTC metadata functions. Wrapper for ExifTool command-line utility. Can read AND write all XMP metadata.

getImageMetadata
public struct getImageMetadata ( required string imageFilePath, boolean exifToolWinsConflict="true" )

Returns all XMP metadata for EXIF and IPTC types. Supplies a merge of metadata from the ColdFusion built-in functions and that from ExifTool.

Output: suppressed
Parameters:
   imageFilePath: string, required, imageFilePath - Absolute file path to the image file.
   exifToolWinsConflict: boolean, optional, exifToolWinsConflict - ColdFusion may see a tag name in a different namespace having a different value than the ExifTool tag namespace. Default is ExifTool wins a conflict. Set to false for ColdFusion to win.

mergeExifToolWithCFFuncs*
private Struct mergeExifToolWithCFFuncs ( required string imageFilePath, required struct exifToolResult, boolean exifToolWinsConflict="true" )

Merges ExifTool output with ColdFusion's built-in ImageGetEXIFMetaData() and ImageGetIPTCMetaData() results. The two sources do not completely overlap. When both return the same tag name from different XMP namespaces, the exifToolWinsConflict flag determines which value is kept.

Output: suppressed
Parameters:
   imageFilePath: string, required, imageFilePath - Absolute file path to the image file.
   exifToolResult: struct, required, exifToolResult - Struct of metadata tags already parsed from ExifTool output.
   exifToolWinsConflict: boolean, optional, exifToolWinsConflict - ColdFusion may see a tag name in a different namespace having a different value than the ExifTool tag namespace. Default is ExifTool wins a conflict. Set to false for ColdFusion to win.
