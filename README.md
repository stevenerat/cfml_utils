# cfml_utils

A collection of ColdFusion utilities written during my years as a technical
support engineer for Adobe ColdFusion. Each utility is standalone and was
built to solve a specific, real-world problem — many were originally
accompanied by blog posts explaining the context and approach.

## Background

These tools were written against **Adobe ColdFusion**. They are not tested
against Lucee, Railo, or other CFML runtimes, and compatibility with those
platforms is not a goal of this project.

Each utility's README documents the specific Adobe ColdFusion version and
OS it was tested on.

## Structure

Each subdirectory is a self-contained utility with its own README covering
purpose, usage, version, and platform notes.

## Status

This repo is an active restoration project. Utilities are being reviewed,
tested against modern Adobe ColdFusion environments, and published
incrementally. Not everything here is production-ready — check individual
READMEs for current status.
